# ESP32-C6-DEV-KIT-N8

ESP32-C6-WROOM-1-N8

* RISC-V 32 单核，160MHz
* 320KB ROM、512KB HP SRAM、16KB LP SRAM、8MB Flash
* WiFi 6、BLE5，IEEE 802.15.4（Zigbee 3.0 & Thread)
